Current projects

Consilium

Development

A personal finance tracker built with Flutter and Firebase. It allows you to track expenses (for example the stuff you bought) and subscriptions (rent, insurance, ...) as well as categorize them as necessary. It provides insight for the user by generating relevant charts and diagrams detailing monthly spending per category.

Consilium Admin CLI

Release

The accompanying administrative tools for the Consilium mobile application to create & delete users as well as migrate, export & import user data. The CLI tool is written using Golang and comes with a few tests for important functions. Its main use is (in addition to providing easier maintenance options for the Firebase database) to give me the option to learn Golang.

Crypto Asset Tracker

Concept

A project to replace my spreadsheet tracking the performance of my cryptocurrency portfolio. The idea is that the user enters the tokens he's holding in his wallets at any given point in time (for example monthly) and the application will return charts, prices and valuations for these wallets. It might also indicate staking performance/returns if the token supports it.

MasterMind

Release

A project created as part of an university assignment. The goal was to create a mobile application where the user can play the tabletop game "MasterMind". The final project supported user management, leader boards, ranking and multiplayer. The backend is written in Nest.js, the frontend in Flutter.
MasterMind TL; DR: A friend (or the computer) comes up with a solution, for example five colors in a specific order. The player has a certain number of guesses to get to the correct solution. After each guess the counterpart will tell the player which colors are in the correct position and which are part of the solution but not yet positioned correctly.
