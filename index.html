Current projects

Horus

Concept

A project to replace the spreadsheet tracking the performance of my cryptocurrency portfolio. The idea is that the user enters the tokens he's holding in his wallets at any given point in time (for example monthly) and the application will return charts, prices and valuations for these wallets. It might also indicate staking performance/returns if the token supports it.

Consilium

Development

A personal finance tracker built with Flutter and Firebase. It allows you to track expenses (for example the stuff you bought) and subscriptions (rent, insurance, ...) as well as categorize them as necessary. It provides insight for the user by generating relevant charts and diagrams detailing monthly spending per category.

Quaesitor

Release

In the context of this term paper, an application was developed that collects data available on the Internet about a person and thus creates a profile based on the defined search parameters. Telephone directories, social networks and sports portals serve as sources. Concepts of the Semantic Web and Knowledge Graphs were used to connect individual data sets. The implementation showed that the quality of the data sources is crucial for the later result. While it turned out that most of the results of the data sources can be put into a standardized format with moderate effort, their quality leaves much to be desired in some places.

MasterMind

Release

A project created as part of an university assignment. The goal was to create a mobile application where the user can play the tabletop game "MasterMind". The final project supported user management, leader boards, ranking and multiplayer. The backend is written in Nest.js, the frontend in Flutter.
MasterMind TL; DR: A friend (or the computer) comes up with a solution, for example five colors in a specific order. The player has a certain number of guesses to get to the correct solution. After each guess the counterpart will tell the player which colors are in the correct position and which are part of the solution but not yet positioned correctly.
